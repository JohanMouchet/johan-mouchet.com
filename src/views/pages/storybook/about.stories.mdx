# About

Design system of [Johan Mouchet's](https://www.johan-mouchet.com/) portfolio.

Open-sourced on [GitHub](https://github.com/JohanMouchet/johan-mouchet.com).

## Getting Started

- Look into the [Settings](/?path=/story/settings-colors--default-story) for an overview of the design tokens pulled from Sass.
- Look into the [Objects](/?path=/story/objects-alert--default-story), [Components](?path=/story/components-projects--default-story), [Partials](?path=/story/partials-header--default-story) and [Layouts](?path=/story/layouts-default--default-story) for an overview of the available components.
  - Use the _Knobs_ tab to edit props dynamically
  - Use the _Docs_ tab dor additional documentation
  - Use the _Viewport_ icon to display a story under a different breakpoint
